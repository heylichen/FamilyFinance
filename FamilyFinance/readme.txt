a web app for testing SSH project

1. details
server: 
   spring 4
   spring mvc 4
   hibernate:4.3
   servlet : 3.0 ,so please run it in a servlet 3 container, eg tomcat 7
   java : 7
js:
   jquery
   d3 
2. note
  this is a maven project, you can import it to eclipse with maven support.
  
  use spring mvc java config instead of web.xml, check 
	   lab.config.spring.WebAppInitializer,
	   lab.config.spring.SpringConfig
  for web configuration details.
  

 
 3. used features 
    spring mvc  multiviews
    spring mvc global exception handler
